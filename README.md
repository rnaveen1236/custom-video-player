# 🎬 Custom Video Player (JavaScript)

A fully functional **Custom HTML5 Video Player** built using **HTML, CSS, and JavaScript** with advanced playback controls and modern UI design.

This project replaces the default browser video controls with a custom-built interactive control system.

---

## 🚀 Project Overview

This custom video player includes:

* ▶️ Play / Pause functionality
* ⏩ Skip Forward & Backward
* 🔊 Volume control with mute toggle
* ⏱️ Real-time video timer
* 🎚️ Interactive timeline with draggable progress bar
* ⚡ Playback speed control (0.5x – 2x)
* 🖥️ Fullscreen mode
* 🪟 Picture-in-Picture (PiP)
* 🎨 Clean responsive UI

## 🧠 How It Works

* JavaScript listens for video events (`play`, `pause`, `timeupdate`)
* Updates progress bar dynamically
* Calculates formatted time (HH:MM:SS)
* Uses DOM manipulation for UI interaction
* Implements fullscreen and PiP using browser APIs
